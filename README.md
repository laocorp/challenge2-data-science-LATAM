Análisis de retención (Churn) para la empresa Telecom X, como parte del Challenge 2 de Data Science LATAM.

Este repositorio contiene:
- El reporte final en formato Jupyter Notebook (`TelecomX_LATAM.ipynb`), donde se ingiere la data desde la API original, se limpia y se analizan las variables principales que impactan en el abandono de los clientes.

***
## Herramientas utilizadas
- **Python**
- **Pandas** para limpieza y manipulación de datos (Data Wrangling y ETL).
- **Plotly** para visualizaciones interactivas.
